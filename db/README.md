
# ForgePLM Database

This folder contains the database definition and static reference data for ForgePLM.

## Structure

- `schema/` — database objects, tables, constraints, indexes, stored procedures, views, functions, and sequences.
- `seed/` — static reference data required for a fresh ForgePLM install.

## Important

Do not commit live operating data, including customers, projects, ECOs, part numbers, revisions, artifacts, file paths, users, or connection strings.
